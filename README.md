# Polkupp

Daglig oppdatert oversikt over Vinmonopolets prisnedsettelser, med korte AI-skrevne omtaler av hver flaske.

## Stack

- React + Vite (frontend)
- Vercel (hosting)
- Supabase (database + Edge Functions)
- Anthropic API (Claude Sonnet med web_search) for omtaler

## Lokal utvikling

```bash
npm install
npm run dev
```
